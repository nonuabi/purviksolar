Security Policy

We take the security of purviksolar.com seriously. If you discover a security vulnerability, we appreciate your help in disclosing it responsibly.

Reporting a Vulnerability

Please report security issues by emailing: purviksolar@gmail.com

Include as much information as possible:

  • URL or endpoint affected
  • Steps to reproduce
  • Expected vs actual behavior
  • Any proof-of-concept code (if available)

Guidelines

  • Do not exploit the vulnerability
  • Do not access or modify user data
  • Do not perform denial-of-service attacks

Response

We aim to acknowledge reports within 72 hours and resolve valid issues as quickly as possible.

Thank you for helping keep our users safe.
